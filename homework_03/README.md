# Homework 03 — Stochastic Processes and Theoretical Foundations

This homework combines machine learning prediction theory with continuous-time stochastic-process methods.

## Topics covered

- Conditional expectation as the optimal predictor under squared loss
- Integrated Wiener process variance
- Random walk approximation to Brownian motion
- Scaling conditions for convergence
- Maximum likelihood estimation for Brownian motion with drift
- Continuous-time model derivations

## Mechanics used

- orthogonality argument for conditional expectation
- expectation/variance computation
- covariance kernel integration
- Itô-style reasoning
- asymptotic scaling of random walks
- likelihood construction from independent increments
- estimator derivation through first-order conditions
