# Homework 01 — Probability and Statistics Foundations

This homework focuses on core probability tools used in machine learning and quantitative modeling.

## Topics covered

- Minimum and maximum of random variables
- Distribution functions of order statistics
- Survival functions
- Hazard rates
- Probability identities involving min/max transformations
- Conditional and marginal reasoning

## Mechanics used

- distribution-function transformations
- independence assumptions
- survival analysis identities
- differentiation of CDFs
- hazard-rate decomposition
