# Homework 02 — Financial Data Analysis and Statistical Diagnostics

This homework applies statistical and machine learning foundations to a real financial time series: the S&P 500.

## Topics covered

- Importing and cleaning market data
- Daily return construction
- Price and return visualization
- QQ-plot analysis
- Shapiro–Wilk normality test
- Jarque–Bera normality test
- Annualized return and volatility
- Period comparison before and after 2000
- Tail-event frequency under normality assumptions

## Mechanics used

- pandas-based time-series processing
- percentage return calculations
- data visualization with matplotlib
- hypothesis testing
- descriptive statistics
- annualization formulas
- comparison of empirical vs theoretical tails
